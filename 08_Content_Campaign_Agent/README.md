## 🎯 What It Does

Turns FLAG's existing blog posts, case studies, and internal notes into **ready-to-review LinkedIn posts, email newsletter drafts, and carousel copy** — delivered daily to Roy or Jessy via Telegram for approval.

**No auto-posting. No spam. Human always reviews first.**

---

## 💼 Value to FLAG

| 🔴 Pain Point | ✅ What This Solves |
|---|---|
| FLAG posts average 4–6 likes — great content, low reach | Consistent weekly posting cadence with varied formats drives compounding reach |
| Writing LinkedIn posts takes time BD doesn't have | Agent drafts 3 post options from any article in 30 seconds |
| No structured content pipeline | Weekly content calendar auto-generated from FLAG's blog |
| BD needs warm leads before cold outreach | Authority content attracts inbound interest from county decision-makers |
| Same message repeated across posts | Agent varies format: text post, carousel, question hook, stat-lead |

---

## ⚙️ How It Works
```
Input: FLAG blog URL or article text
              ↓
🔍 Agent reads + summarizes key insights
              ↓
✍️  Generates:
    - 3 LinkedIn post drafts (hook-first, CTA-last)
    - 1 carousel outline (5–7 slides)
    - 1 email snippet for BD outreach
              ↓
📱 Delivered via Telegram to Roy/Jessy for review
              ↓
👤 Human edits → posts manually (no auto-publishing)
```

## 🛠️ Tech Stack

| Layer | Technology | Purpose |
|---|---|---|
| **Content Drafting** | OpenAI GPT-4o | Generate post drafts, carousels, email copy |
| **Article Ingestion** | BeautifulSoup | Scrape FLAG blog articles automatically |
| **Backend** | FastAPI | API trigger for manual or scheduled runs |
| **Delivery** | OpenClaw → Telegram | Daily digest to Roy/Jessy for approval |
| **Calendar Output** | CSV / Google Sheets | 4-week content plan export |
