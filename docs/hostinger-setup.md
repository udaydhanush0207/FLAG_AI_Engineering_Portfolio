# Hostinger KVM1 Setup

## Create Server
1. Go to [hostinger.com](https://hostinger.com) → VPS Hosting
2. Choose KVM1 plan ($4.99/mo)
3. Select Ubuntu 24.04
4. Choose closest datacenter
5. Copy IP address and root password

## Cost
- $4.99/month
- Monthly billing

## Connect
```bash
ssh root@YOUR_IP
```

Then follow the [main setup guide](./setup-guide.md).
